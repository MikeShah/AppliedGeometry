# Readme

This serves as a 'template' project that you can learn from or otherwise use in this course.

## To Run

- Ensure that you have the 'dmd' compiler installed
  - `dmd --version` can test.
- You can then simply run `dub` to build and run the application.
- If it does not work, you are likely missing the 'SDL3' library.
  - See the playlist here: https://www.youtube.com/playlist?list=PLvv0ScY6vfd-5hY-sFyttTjfuUxG5c7OA
